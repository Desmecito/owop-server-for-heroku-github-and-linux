Instructions for heroku


1. Do account on: https://heroku.com , https://netlify.com , and https://github.com (if you dont have)

2. Download heroku cli , and git

3. Do heroku app. In settings (in heroku app) set buildpack to http://github.com/heroku/heroku-buildpack-c.git

4. Download this repository and that https://github.com/mathmakgakpak/owopppa to your pc and unpack

5. Edit world.cpp and other things

6. Push server (owop-server-for-heroku-github-and-linux) to heroku using deploy inscrutions ( https://dashboard.heroku.com/apps/yourapp/deploy/heroku-git )

7. Now edit client ( https://github.com/mathmakgakpak/owopppa ) ask me how to edit (mathias377)

8. Push client to netlify.com and play on your owop if you want owop bot go to https://github.com/mathmakgakpak/owop-to-discord-bot-OJS
